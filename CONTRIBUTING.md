# Contributing to OpenAnalytics

Thanks for your interest in contributing!

## Getting Started

```bash
git clone https://github.com/Acidias/OpenAnalytics.git
cd OpenAnalytics
npm install
docker compose up -d    # Start PostgreSQL + Redis
npm run dev             # Start development servers
```

## What to Work On

Check [issues](https://github.com/Acidias/OpenAnalytics/issues) for open tasks. Good first issues are tagged with `good-first-issue`.

**Priority areas:**
- Tracker script size optimization
- Dashboard components
- API performance
- Documentation and examples

## Code Style

- TypeScript everywhere
- Prettier for formatting
- ESLint for linting
- Write tests for new features

## Pull Requests

1. Fork the repo
2. Create a branch from `main`
3. Make your changes
4. Run `npm run test` and `npm run lint`
5. Submit a PR with a clear description

Keep PRs focused — one feature or fix per PR.

## Questions?

Open an issue or start a discussion. We're friendly.
