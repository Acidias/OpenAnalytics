# @openanalytics/shared

Shared TypeScript types, validation schemas, and utilities used across all packages.

## Contents

- Type definitions for tracking events, analytics data, API responses
- Zod schemas for request/response validation
- Common constants (device breakpoints, country codes)
- Error types

## Usage

```typescript
import { TrackingEvent, AnalyticsQuery, SiteConfig } from '@openanalytics/shared';
```
